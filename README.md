# Sistema de Biblioteca SGBD

Proyecto en Python que simula un sistema de gestión de biblioteca.

## Funcionalidades
- Gestión de libros (físicos y digitales)
- Registro de usuarios (Alumno, Profesor, Administrador)
- Sistema de préstamos y devoluciones
- Persistencia en JSON
- Reportes y estadísticas

## Tecnologías
- Python
- Programación Orientada a Objetos
- JSON

## Ejecución
```bash
python main.py
